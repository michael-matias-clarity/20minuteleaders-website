# 20MinuteLeaders Website Rebuild

## Mission
Rebuild 20minuteleaders.com with our authentic brand while incorporating premium features inspired by 20VC.

## Reference: 20VC Features
From thetwentyminutevc.com episode pages:
- Guest bio prominently displayed
- Structured episode outline (discussion topics)
- Key questions/talking points as headers
- Related episodes
- Subscribe CTAs
- Clean, professional design

## Current Assets
- **Episodes data:** 1,171+ episodes in episodes.json
- **Transcripts:** 10 pilot episodes transcribed (ep1162-1171)
- **Audio:** Links to Spotify/Anchor
- **RSS Feed:** https://anchor.fm/s/ce59eb28/podcast/rss
